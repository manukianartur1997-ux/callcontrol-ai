## Shared checklist — перед каждой отправкой
- [ ] Имя/роль current в LinkedIn (профиль активен за 3–6 мес).
- [ ] Канал реально открыт (LinkedIn принимает DM / email валиден через футер/hunter.io / Telegram доступен).
- [ ] Падеж имени в обращении верный (Вадиме / Ілля / Руслане / Максиме / Антоне).
- [ ] Сообщение без claims о просмотре звонков — только «перевірити релевантність». ✓
- [ ] Laba + robot_dreams = один человек (Рейніш) — не дублировать.
- [ ] Лог в CSV: status `contacted` + дата; ответы и возражения — дословно в notes.

## Cadence
День 0 — отправить все 5 (LinkedIn). День 3–4 — follow-up по неответившим. День 7 — fallback-канал. Один follow-up на канал, без давления.
